---
title: Member Punishment Key
description: This punishment key is useful if you need to find out what to do when a member breaks a rule.
---

| Rule Broken                                                  | Action to take                                                    | When to enforce                                                                                                                                         |
| ------------------------------------------------------------ | ----------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| No livery                                                    | <span className="text-green-400 font-semibold">Verbal warn</span> | Using the default ERLC livery instead of the in-game liveries provided.                                                                                 |
| No uniform or Incorrect uniform                              | <span className="text-green-400 font-semibold">Verbal warn</span> | Not wearing the correct uniform as a Police Officer, Sheriff, FD, or DOT.                                                                               |
| Random Deathmatch (RDM)                                      | <span className="text-green-400 font-semibold">Warn</span>        | If a random player kills them for no reason.                                                                                                            |
| Vehicle Deathmatch (VDM)                                     | <span className="text-blue-400 font-semibold">Warn</span>         | Using their vehicle to purposely run into other vehicles or players with malicious intent and for no reason. Damaging a vehicle randomly for no reason. |
| New Life Rule (NLR)                                          | <span className="text-blue-400 font-semibold">Warn</span>         | Failing to roleplay an entirely new life after dying.                                                                                                   |
| Fear Roleplay                                                | <span className="text-blue-400 font-semibold">Warn</span>         | The concept of their character being afraid to die.                                                                                                     |
| Using a banned or whitelisted only weapon (VIP/BOOSTER ONLY) | <span className="text-blue-400 font-semibold">Warn</span>         | Using weapons that you are not allowed to use. (Boosters and VIPs are allowed to use Sniper, shotgun, and colt.)                                        |
| Wearing an unrealistic avatar                                | <span className="text-blue-400 font-semibold">Warn</span>         | Anything that you would not see in real life. This involves wearing cat ears or furry items while on a law Enforcement team.                            |
| Interfering with a mod scene                                 | <span className="text-blue-400 font-semibold">Warn</span>         | Disrupting with an on-going mod scene after being told to leave.                                                                                        |
| Fail Roleplay                                                | <span className="text-blue-400 font-semibold">Warn</span>         | When a person fails to roleplay properly.                                                                                                               |
| Trolling                                                     | <span className="text-yellow-400 font-semibold">Kick</span>       | If you're unsure on what is considered trolling feel free to ask another staff member or higher up.                                                     |
| Staff Impersonation                                          | <span className="text-yellow-400 font-semibold">Kick</span>       | Wearing a staff uniform, staff livery, and opposing as a staff member.                                                                                  |
| Staff Evasion                                                | <span className="text-yellow-400 font-semibold">Kick</span>       | Fleeing/running away from staff to avoid punishment.                                                                                                    |
| 3 warnings                                                   | <span className="text-yellow-400 font-semibold">Kick</span>       | Be aware all 3 warnings have to be on the same day.                                                                                                     |
| Leaving to avoid punishment (LTAP)                           | <span className="text-red-600 font-semibold">Ban</span>           | Leaving the game to avoid a punishment.                                                                                                                 |
| NSFW                                                         | <span className="text-red-600 font-semibold">Ban</span>           | Adult Content 18+ If you're unsure on what is considered NSFW feel free to ask another staff member or higher up.                                       |
| Breaking Roblox TOS                                          | <span className="text-red-600 font-semibold">Ban</span>           | If you're unsure on what is considered against Roblox TOS feel free to ask another staff member or higher up.                                           |
| Mass Vehicle Deathmatch (VDM)                                | <span className="text-red-600 font-semibold">Ban</span>           | Using their vehicle to purposely run into other 5+ vehicles or 5+ players with malicious intent and for no reason.                                      |
| Mass Random Deathmatch (RDM)                                 | <span className="text-red-600 font-semibold">Ban</span>           | Killing 5 people or more randomly would be considered mass RDM.                                                                                         |
| Racism/Homophobia                                            | <span className="text-red-600 font-semibold">Ban</span>           | If you're unsure on what is considered racism/homophobia feel free to ask another staff member or higher up.                                            |
